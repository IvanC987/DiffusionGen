Help & Documentation

Your guide to understanding settings and prompt creation.

Basic Settings

  • Prompt: Enter a descriptive text to guide AI image generation.
  • CFG Scale: Controls how strictly the AI follows the input prompt.
    • Lower values (1-4) allow more randomness.
    • Higher values (7-12) make the AI adhere strictly to the prompt.
  • Batch Size: Number of images generated per request.
  • Denoising Steps: Number of noise-removal iterations.
    • DDPM: Recommended: 1000 steps.
    • DDIM: Recommended: 20-40 steps.

Advanced Settings

  • Sampling Method:
    • DDIM: Faster sampling, fewer steps.
    • DDPM: More stable, requires more steps.
  • Real-ESRGAN: AI upscaling for sharper images.
    • 2x: Doubles resolution.
    • 4x: Quadruples resolution.
  • Real-Time Denoising: View denoising progress at each step.

Image Modifications

  • Upload Image (img2img): Modify an uploaded image.
  • Strength (img2img): Affects how much AI modifies the input.
    • 0.1 - 0.4: Minor changes.
    • 0.5 - 0.7: Balanced modifications.
    • 0.8 - 0.9: Heavy modifications.
  • Upload Mask (Inpainting): Selective editing within masked areas.

Random Prompt Generator

Generate AI-generated prompts for inspiration.

  • Select a category (e.g., Animals, Humans, Scenery).
  • Click Generate Random Prompt.
  • If no category is selected, an error appears.

Prompt Creation Tips

  1. Use Keywords: Stick to known model words.
  2. Structured Descriptions: Use Stable Diffusion-style formatting.
  3. Balance Specificity: Avoid overly complex descriptions.
  4. Use Sensory Details: Mention lighting, moods, textures.
  5. Experiment with Styles: Try "cinematic lighting," "watercolor art," etc.

Example Training Prompts:

Beneath towering, snow-capped peaks, a majestic tiger leaps...
Golden afternoon light bathes a tranquil pond...
Moonlight filters through dense reeds of a misty wetland...
Vivid hues of orange and pink paint the dusky sky.
